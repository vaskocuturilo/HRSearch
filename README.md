# HRSearch
SmtpServer and SmtpClient. 


Smtp Server 
=================================================================================================================================
For start Smtp Server Usage : 
 - sudo python smtpserver --start 
 For help Usage: 
 - sudo python smtpserver --help 
=================================================================================================================================


Smtp Client 
=================================================================================================================================
For start Smtp client Usage: 
 - python smtpclient --start 
 For help Usage: 
   python smtpclient --help
